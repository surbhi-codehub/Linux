# Title: Creating Files with touch

## 1. Go to your linux_practice directory and create an empty file named notes.txt.
### Ans  cd ~/linux_practice && touch notes.txt
---
## 2. Create three files named app.py,query.sql, and README.md using a single touch command.
### Ans  touch app.py query.sql README.md
---
## 3. List all files current directory to verify they were created.
### Ans  ls
---
## 4. Create a new directory named day04 and inside it create a file named practice.txt.
### Ans  mkdir day04 && cd day04 && touch practice.txt
---
## 5. Display your current working directory after creating all files.
### Ans  pwd
