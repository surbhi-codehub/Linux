# 🐧 100 Days of Linux Challenge

Welcome to my **100 Days of Linux** learning journey.

This repository contains my daily Linux practice while learning **Ubuntu on WSL (Windows Subsystem for Linux)** from absolute beginner to writing practical shell scripts.

The goal of this challenge is to build a strong Linux foundation that will help in Software Development, Python, SQL, Data Engineering, Cloud, and DevOps.

---

# 📅 Challenge Duration

- Total Days: **100**
- Daily Practice: **5 Hands-on Questions**
- Platform: **Ubuntu (WSL)**
- Repository Type: **Daily Practice Journal**

---

# 🎯 Goals

By the end of this challenge, I will be able to:

- Navigate the Linux file system confidently
- Manage files and directories
- Search files and text efficiently
- Understand Linux permissions
- Work with processes
- Monitor system resources
- Use networking commands
- Manage packages
- Write Bash scripts
- Automate repetitive tasks
- Build confidence using the Linux terminal
